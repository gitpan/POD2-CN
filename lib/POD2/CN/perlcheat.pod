=head1 NAME

perlcheat - Perl 5 速查

=head1 DESCRIPTION

这个“速查”实际上是一个真正的“手册”－－巴掌大的一个小册子，用来记录
一些开始学习 Perl 编程时遇到的知识点。它并不是面面俱到的，但是覆盖了约
194 个 Perl 特性。
